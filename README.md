# Python Backend service Template

## Prerequisite
- python
- pyenv (opt)
- pipenv

## Installation
`setup your .pypirc`
`mkdir .venv`
`pipenv install`

## Run ( Check Pipfile )
- dev
`pipenv run dev`

- prod test
`pipenv run prod`

## How to use 
- Clone or Fork this project for a new one
- Design your config in configurations
